Prompts

Six reusable prompts cover the common jobs. Drop one into Claude Code with the skill installed, or just describe your task in natural language. Source: claude/skills/dga-platforms-code/prompts/.

The prompt set

#PromptUse it to…
01Audit existing UIAudit a live interface for alignment with Platforms Code principles.
02Redesign existing UIRedesign an existing page while preserving logic and APIs.
03Build a new pageBuild a new page from scratch in the correct style.
04Convert to Platforms Code styleConvert an interface to the Platforms Code visual language.
05Review Arabic RTLReview RTL correctness and Arabic content quality.
06Final quality gateRun the pre-release quality check before declaring done.

Example invocations

With the skill installed, natural language is enough — Claude activates the skill from the task context.

# Build
Build an Arabic RTL service landing page for "إصدار شهادة"
aligned with DGA Platforms Code. UI only — don't touch APIs or auth.

# Audit
Audit this dashboard against DGA Platforms Code: tone, RTL, accessibility,
and component usage. List findings and manual-review items.

# Convert
Convert this React page to the Platforms Code style using the verified
tokens and the bundled component library. Keep validation behaviour.

# RTL review
Review this page for Arabic RTL correctness: mirrored icons, logical CSS,
and number/code isolation. Report what must change.
💬

Each task ends with the skill's required report: scope, changed files, what was preserved (logic/APIs/auth/validation), checks run, manual-review items, and a compliance note. See the Overview.

Aligned with Platforms Code principles — not official DGA compliance. Official source: design.dga.gov.sa.
