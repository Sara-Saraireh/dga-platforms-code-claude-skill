Component library · increment 1

Components

Eight generic, reusable React + plain-CSS components aligned with Platforms Code component anatomy and driven entirely by the verified tokens. Each preview below is real DOM styled by the skill's own tokens.css and components.css. Use the RTL ↔ LTR button to see mirroring.

⚖️

Illustrative, not official. These are product-agnostic components aligned with Platforms Code principles — not official DGA Figma/Storybook assets, and they assert no official compliance. They copy no official code, assets, or fonts. Load IBM Plex Sans Arabic in your host project.

import "dga-platforms-code/components/tokens.css";
import "dga-platforms-code/components/components.css";
import { Button, Input, Alert, Stepper } from "dga-platforms-code/components";

Button

Aligned with «الأزرار». One primary action per view; secondary/tertiary are quieter. Renders a real <button> with a visible focus ring and a 44px touch target.

Button — variants, sizes & states
PropTypeDefaultNotes
variantprimary · secondary · tertiary · destructiveprimaryVisual style.
sizesm · md · lgmdHeight 36 / 44 / 48px.
blockbooleanfalseFull-width button.
leadingIcon / trailingIconnodenullDecorative icons (aria-hidden).
disabledbooleanfalseDisabled state.
 
<Button variant="primary">ابدأ الخدمة</Button>
<Button variant="secondary">حفظ كمسودة</Button>
<Button variant="destructive" size="sm">حذف</Button>

Badge / Tag

Aligned with «العلامات». Status uses text (and an optional dot) — never color alone.

Input

Aligned with «المدخلات». The label is always visible (placeholders are never labels). Errors are announced (role="alert") and linked to the field.

Input — default, hint, required & error

مثال: 12345678

صيغة البريد الإلكتروني غير صحيحة.

PropTypeNotes
labelstringAlways rendered, linked to the input.
hintstringHelper text, linked via aria-describedby.
errorstringSets aria-invalid and announces the message.
requiredbooleanShows * + screen-reader “(مطلوب)”.
optionalTextstringDefaults to (اختياري).
 
<Input label="رقم الطلب" hint="مثال: 12345678" />
<Input label="البريد الإلكتروني" required
       error="صيغة البريد الإلكتروني غير صحيحة." />

Alert

Stepper

Aligned with «الخطوات». Steps flow right-to-left in document order; the current step is conveyed via aria-current="step". States: complete · current · upcoming.

Stepper — 3 steps, current = 2
  1. ✓البيانات
  2. 2المراجعة
  3. 3الإرسال
<Stepper current={1}
  steps={[{ label: "البيانات" }, { label: "المراجعة" }, { label: "الإرسال" }]} />

Breadcrumbs

Aligned with «مسار التصفح». Wrapped in a labelled <nav>; the last item is the current page. Separators sit in the RTL flow and mirror automatically.

Breadcrumbs
  1. الرئيسية/
  2. الخدمات/
  3. إصدار شهادة
<Breadcrumbs items={[
  { label: "الرئيسية", href: "/" },
  { label: "الخدمات", href: "/services" },
  { label: "إصدار شهادة" },
]} />

Modal

Aligned with «النوافذ المنبثقة». A labelled dialog with a focus trap while open, focus restore on close, and Escape to dismiss. The static preview below shows the dialog surface.
