Component library · increment 2

Charts & data visualization

Ready-to-use, dependency-free chart components — plain SVG/CSS, no charting library required. They match the official DgaChart (platformscode-react) palette exactly — a monochromatic green scheme (no gold or lavender for data series) with a neutral gray for any remaining series. They never distinguish data by color alone, read right-to-left, and require a text summary for screen readers. Pie/donut supports up to six categories.

Official DgaChart color arrays: donut #1B8354 · #079455 · #B8EACB · #54C08A · line #1B8354 · #079455 · bar #1B8354 · #54C08A · #E5E7EB. The skill exposes these as --pc-chart-series-1…4 + --pc-chart-neutral.

⚖️

Illustrative, not official. Aligned with the Platforms Code charts page (pie/donut, line, column/bar) — not official DGA assets, and no compliance claim. No invented palette; labels, legend, and dashes carry meaning.

import "dga-platforms-code/components/tokens.css";
import "dga-platforms-code/components/components.css";
import "dga-platforms-code/components/charts/charts.css";
import { ChartContainer, LineChart, BarChart, DonutChart, KpiCard } from "dga-platforms-code/components";

KpiCard

A single-number indicator: label → value → trend → context. The trend is text + icon, never color alone.

LineChart

Trends over time. The time axis reads right-to-left (first period on the right). Up to two series; the second is dashed so it is distinguishable without color. Shown inside a ChartContainer with its required accessible summary, legend, and source.

LineChart in ChartContainer
<ChartContainer
  title="عدد المعاملات المعالَجة شهريًا"
  description="المعاملات المعالَجة مقابل قيد المراجعة، يناير–يونيو."
  summary="ارتفع عدد المعاملات المعالَجة من 1,500 في يناير إلى 2,400 في يونيو."
  source="المصدر: بيانات تجريبية"
  legend={<><span><i className="pc-chart__swatch"
    style={{ background: "var(--pc-chart-series-1)" }} /> معالَجة</span></>}
>
  <LineChart data={[
    { period: "يناير", series1: 1500, series2: 300 },
    { period: "يونيو", series1: 2400, series2: 275 },
  ]} />
</ChartContainer>

BarChart

Categorical comparison. Use orientation="horizontal" for many categories or long Arabic labels (labels get full width and are never truncated). RTL: first category on the right (vertical) or on top (horizontal).

BarChart — vertical
BarChart — horizontal (long Arabic labels)

  { category: "تجديد ترخيص ساري", value: 2750 },
  { category: "الاعتراض على قرار", value: 760 },
]} />

DonutChart

Simple part-to-whole for a few segments. A label — value (%) legend sits next to the ring so segments are never distinguished by color alone.

DonutChart with legend

Chart states

Every chart ships calm, accessible empty / loading / error states via the ChartContainer state prop (ChartEmpty, ChartLoading, ChartError).

Empty · Loading · Error
<ChartContainer title="الطلبات الشهرية" state="loading" />
<ChartContainer title="الطلبات الشهرية" state="empty"
  emptyHint="ستظهر البيانات هنا بمجرد توفّرها." />
<ChartContainer title="الطلبات الشهرية" state="error" onRetry={refetch} />
